Mushroom Information System

육각형 격자 기반 웹 버섯 수집 게임

공공데이터와 게임의 만남

🎯 프로젝트 개요

게임 + 교육 + 실제 데이터

산림청 국립수목원의 실제 버섯 데이터를 활용한 교육용 웹 게임


🎮 재미있는 게임플레이
육각형 타일 탐험과 수집
📚 교육적 가치
실제 버섯 생태 정보 학습
🌐 웹 기반
브라우저에서 바로 플레이
👥 멀티플레이어
실시간 경쟁 모드 지원

🎮 게임플레이

플레이 방식

1. 탐험

육각형 타일을 클릭하여
새로운 버섯 발견

2. 학습

버섯의 환경별 분포
패턴 학습

3. 수집

다양한 버섯 수집으로
점수 획득

4. 경쟁

다른 플레이어와
실시간 경쟁
